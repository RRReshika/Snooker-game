# Premium Snooker Game

A professional snooker simulation built with p5.js and Matter.js.

## Project Structure

```
Snooker-game/
├── index.html              # Main game page
├── home.html              # Landing page
├── extensions.html        # Documentation page
├── style.css              # Game page styles
├── home-styles.css        # Landing page styles
├── extensions-styles.css  # Documentation styles
├── sketch.js              # Main game logic (p5.js + Matter.js)
├── home-script.js         # Landing page animations
└── extensions-script.js   # Documentation interactions
```

## Features

### Game Mechanics
- Realistic Physics using Matter.js engine
- 3D Camera with manual and auto-snap controls
- Multiple setup modes (Triangle, Random, Practice)
- Two rule sets (Standard and Beginner)

### Visual Features
- 5 table colors to choose from
- Shot replay system
- Slow motion playback
- Animated loading screen

### Player Features
- Individual player scores and break counting
- Turn indicators
- Automatic foul detection
- Frame and shot counter
