3D Snooker Experience

Extensions & Extra Features

Enhanced gameplay mechanics and broadcast-quality presentation

01

Advanced Physics Engine

What Was Added

Integration of Matter.js physics library to simulate realistic ball dynamics including friction, collision response, momentum transfer, and spin mechanics.

Why It Improves Gameplay

Provides authentic ball behavior matching real snooker physics. Players experience realistic cushion bounces, ball-to-ball collisions, and natural deceleration, making strategic shot planning essential.

Implementation Details

  • Realistic coefficient of restitution for cushions and balls
  • Angular and linear velocity calculations
  • Precise collision detection and response
  • Gradual friction-based deceleration
02

Professional Aiming System

What Was Added

Visual aiming guide with dynamic cue positioning, power meter, and trajectory prediction to assist shot planning.

Why It Improves Gameplay

Enables players to visualize shot angles and power requirements. The aiming system replicates the mental calculations professional players make, making the game accessible yet strategically deep.

UI Components

  • Rotatable cue stick following mouse position
  • Power bar with visual feedback
  • Trajectory line showing predicted ball path
  • Real-time angle calculations
03

Broadcast-Style Scoring

What Was Added

Comprehensive scoring system tracking player points, break scores, frame progress, and foul detection with automatic point allocation.

Why It Improves Gameplay

Maintains official snooker rules for scoring, creating an authentic competitive experience. Break tracking encourages strategic play and rewards consecutive successful pots.

Features

  • Live score tracking for two players
  • Break counter showing current run
  • Automatic foul penalty calculation
  • Shot outcome feedback system
04

Official Rules Engine

What Was Added

Complete implementation of World Snooker Tour official rules including legal shot validation, ball-on determination, foul detection, and game state management.

Why It Improves Realism

Enforces authentic snooker rules: red-color alternation, final colors sequence, touching ball rules, and various foul conditions. Creates genuine match scenarios.

Rules Enforced

  • Ball-on determination (red/color alternation)
  • Multiple foul types (miss, wrong ball, in-off)
  • Final colors sequence after reds cleared
  • Minimum 4-point foul penalty
  • Touching ball placement rules
05

Broadcast Interface Design

What Was Added

Professional television broadcast-inspired UI with player panels, status displays, match progress indicators, and cinematic visual elements.

Why It Enhances Experience

Creates immersive viewing experience similar to watching professional snooker on television. Clean, minimal design keeps focus on gameplay while providing essential information.

Interface Elements

  • Player score panels with active player highlighting
  • Status panel showing game mode and current action
  • Match progress tracker (frame, balls remaining, shots)
  • Shot feedback messaging system
  • Subtle animations and transitions
06

Multiple Game Modes

What Was Added

Selectable game modes including Standard Snooker, Practice Mode, and Quick Match variants with different rule sets and time constraints.

Why It Improves Usability

Accommodates different skill levels and play preferences. Practice mode allows learning without pressure, while standard mode provides full competitive experience.

Available Modes

  • Standard Mode: Full official rules enforcement
  • Practice Mode: Relaxed rules for skill development
  • Quick Match: Shortened frames for faster gameplay
  • Mode selection via dropdown interface
07

Automated Ball Management

What Was Added

Automatic ball respawning system with visual dispenser effect. Colors return to their spots after being potted (during red phase), maintaining official snooker spot positions.

Why It Improves Gameplay

Eliminates manual ball replacement, maintaining game flow. The visual dispenser adds satisfying feedback while respecting official spot positions for each color ball.

System Features

  • Animated ball respawn from dispenser tube
  • Correct spot position for each color
  • Automatic cue ball replacement after fouls
  • Visual confirmation of ball state changes
08

Board Colour Customisation

What Was Added

A premium colour selector allowing players to switch the table felt between five preset options: Green (default), Red, Black, Blue, and Pink. Changes apply instantly with smooth colour transitions.

Why It Improves Personalisation

Enables players to customise their playing environment while maintaining professional aesthetics. Different colours can reduce eye strain during extended sessions and add variety to the visual experience without compromising realism.

Features & Implementation

  • Five professionally selected felt colours with proper contrast
  • Smooth fade transitions between colour changes (0.05 per frame)
  • Selective colouring - only affects table felt, preserves ball and UI colours
  • Adaptive cushion and edge colouring for realistic appearance
  • Non-intrusive settings panel in bottom-right corner
  • Premium button design matching broadcast theme
  • Instant application without game restart
  • Maintains proper lighting and shadow effects across all colours
